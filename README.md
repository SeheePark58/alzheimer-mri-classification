\# Alzheimer MRI Classification



This project classifies Alzheimer's disease stages using brain MRI images and a lightweight CNN model.



\## Project Overview



\- Dataset: Brain MRI Alzheimer's dataset

\- Task: Image classification

\- Classes: Non Demented, Very Mild Demented, Mild Demented

\- Model: Lightweight CNN

\- Main file: `alzheimer\_classification.ipynb`



\## Results



The project includes:



\- Class distribution visualization

\- Training and validation curves

\- Confusion matrix

\- Grad-CAM visualization



\## Files



\- `alzheimer\_classification.ipynb`: Main training and evaluation notebook

\- `class\_distribution.png`: Dataset class distribution

\- `training\_curves.png`: Training/validation performance curves

\- `confusion\_matrix.png`: Confusion matrix result

\- `gradcam\_results.png`: Grad-CAM visualization

\- `sample\_images.png`: Sample MRI images
